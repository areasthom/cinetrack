# 🎬 CineTrack

Sistema de avaliação de filmes desenvolvido em Java, com foco em Programação Orientada a Objetos (POO) e modelagem de regras de negócio.

---

## 🚀 Sobre o projeto

O CineTrack é um sistema console onde usuários podem registrar filmes assistidos, avaliar com notas e escrever resenhas.

O sistema também possui lógica de progressão de usuário baseada na quantidade de avaliações realizadas.

---

## 🧠 Funcionalidades

- 👤 Cadastro de usuários
- 🎬 Registro de avaliações de filmes
- ⭐ Sistema de notas e resenhas
- 📊 Cálculo de média de avaliações
- 🏆 Ranking de filmes (Top 3)
- 📈 Sistema de níveis de usuário (gamificação)

---

## 💻️ Estrutura do projeto

O projeto segue uma estrutura simples de camadas:

- `model` → entidades do sistema (Filme, Usuario, Avaliacao)
- `service` → regras de negócio (SistemaCineTrack)
- `app` → interface do usuário (Main)

---

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Separação de responsabilidades
- Modelagem de domínio
- Estruturas de dados (ArrayList)
- Lógica de negócios
